곰코노미 개인정보 처리 안내

본 안내는 학교 수업(교실경제 활동) 운영을 위해 곰코노미에서 처리되는 개인정보 항목과 절차를 설명합니다.

1. 최소 처리 원칙 준수

  • 수집 항목: 학생 번호, PIN, (선택) 이름, 포인트/거래 등 학습활동 데이터
  • 수집 목적: 학생별 교실경제 활동 관리, 포인트·쿠폰·주식 거래 기록 제공, 교육적 피드백
  • 보유 기간: 해당 학년도 종료 시까지(종료 후 삭제 또는 익명화)

2. 개인정보 안전조치 의무

  • HTTPS 통신 사용
  • 교사 PIN 기반 관리자 기능 제한
  • 식별정보 최소화(번호 중심 운영), 민감정보 미수집
  • 학년도 종료 후 데이터 삭제 원칙

3. 열람·정정·삭제·처리정지 절차

  • 학생 및 법정대리인은 언제든 본인 정보의 열람/정정/삭제/처리정지를 요청할 수 있습니다.
  • 요청 방법: 담임교사에게 요청 → 교사가 관리자 화면에서 조치

4. 만 14세 미만 아동 보호

  • 학기 초 안내 및 (필요 시) 법정대리인 동의 절차를 진행합니다.
  • 교육 목적 외 활용을 금지합니다.
